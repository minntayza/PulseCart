# FeedbackPanel UI Design Contract

> Visual and interaction contract for updating the FeedbackPanel component. Generated by gsd-ui-researcher, verified by gsd-ui-checker.

---

## Current State Analysis

### What's Wrong

| Issue | Location | Detail |
|-------|----------|--------|
| UserId displayed prominently | Line 105 | `msg.userId` shown as bold primary text — exposes internal IDs, adds visual noise |
| Font sizes too small | Entire component | All text uses `text-xs` (12px) or `text-[10px]` — below comfortable reading threshold |
| Low-opacity body text | Lines 86, 109 | `text-text/70` (70% opacity) — poor contrast, especially in dark theme |
| Card backgrounds inconsistent | Lines 75, 103 | `bg-white/5` is semi-transparent white — works in dark mode but looks washed out compared to other panels using `bg-surface-alt` or `bg-surface` |
| No timestamp shown | Line 109 | `msg.createdAt` exists in the type but is never rendered — loses temporal context |
| Theme badge too small | Line 106 | `text-[10px] px-1.5 py-0.5` — nearly unreadable at that size |
| Severity badge cramped | Line 107 | Same tiny sizing as theme badge |
| Inconsistent card padding | Lines 75, 103 | Theme cards use `p-3`, message cards use `p-2` — should match |
| No visual separation between sections | Line 57 | `space-y-4` is minimal spacing between Themes and Recent Feedback sections |
| Empty states too terse | Lines 91, 113 | One-liner text with no visual affordance |

### Alignment with Other Dashboard Panels

| Pattern | OrdersPanel | MarketPanel | FeedbackPanel (current) |
|---------|-------------|-------------|------------------------|
| Card padding | `p-3` | `p-3` | `p-2` (messages), `p-3` (themes) |
| Card bg | `bg-white/5` | `bg-white/5` | `bg-white/5` |
| Primary text | `text-sm font-medium` | `text-sm font-medium text-text` | `text-xs font-medium text-text` (themes), `text-xs font-medium text-text` (messages) |
| Secondary text | `text-xs text-muted` | `text-xs text-muted` | `text-xs text-muted` (themes), `text-text/70` (messages — inconsistent) |
| Badge sizing | `text-[10px] px-2 py-1` | `text-[10px] px-2 py-1` | `text-[10px] px-1.5 py-0.5` (smaller than peers) |

## Design Contract

### Design System

| Property | Value |
|----------|-------|

### Spacing Scale

Declared values (multiples of 4):

| Token | Value | Usage |
|-------|-------|-------|
| xs | 4px | Icon gaps, inline badge padding |
| sm | 8px | Compact element spacing, message card internal gaps |
| md | 16px | Card padding, section spacing |
| lg | 24px | Section breaks (between Themes and Recent Feedback) |
| xl | 32px | Not used in this component |

Exceptions: none

---

### Typography

| Role | Size | Weight | Line Height | Tailwind Class |
|------|------|--------|-------------|----------------|
| Section heading | 12px | 600 (semibold) | 1.5 | `text-xs font-semibold uppercase tracking-wider` |
| Card title | 14px | 500 (medium) | 1.4 | `text-sm font-medium` |
| Card body | 14px | 400 (regular) | 1.5 | `text-sm` |
| Badge text | 11px | 500 (medium) | 1.0 | `text-[11px] font-medium` |
| Timestamp | 12px | 400 (regular) | 1.4 | `text-xs` |

**Key change:** All body text moves from `text-xs` (12px) to `text-sm` (14px). Badge text moves from `text-[10px]` to `text-[11px]`.

---

### Color

| Role | Value | Usage |
|------|-------|-------|
| Dominant (60%) | `var(--background)` / `#f7f8fc` | Page background |
| Secondary (30%) | `var(--surface)` / `#ffffff` | Card backgrounds (change from `bg-white/5` to `bg-surface`) |
| Accent (10%) | `var(--primary)` / `#5b5bd6` | Analyze button, active tab indicator |
| Destructive | `var(--danger)` / `#ef4444` | High severity badges only |

**Accent reserved for:** Analyze Now button background, active severity badge (high). NOT for theme badges or general interactive elements.

**Card background change:** Replace `bg-white/5` with `bg-surface-alt` (`#f3f4f8` light / `#1c202b` dark) for better contrast against the page background, matching how StatsRow cards use `bg-surface` with a border.

## Specific CSS Class Changes

### Theme Cards (Identified Themes section)

**Current (line 75):**
```tsx
<div key={i} className="p-3 bg-white/5 rounded-lg">
```

**After:**
```tsx
<div key={i} className="p-3 bg-surface-alt rounded-lg border border-border-light">
```

**Rationale:** `bg-surface-alt` provides solid contrast. `border-border-light` adds subtle definition matching StatsRow card pattern.

### Theme Card Inner Layout (line 76-87)

**Current:**
```tsx
<div className="flex items-center justify-between mb-1">
  <div className="flex items-center gap-2">
    <span>{themeIcons[theme.theme] || '📝'}</span>
    <span className="text-sm font-medium text-text">{theme.theme}</span>
    <span className="text-xs text-muted">({theme.messageCount} mentions)</span>
  </div>
  <span className={`text-[10px] font-medium px-2 py-1 rounded-full ${severityColors[theme.severity]}`}>
    {theme.severity}
  </span>
</div>
<p className="text-xs text-text/70 ml-6">{theme.fixSuggestion}</p>
```

**After:**
```tsx
<div className="flex items-center justify-between mb-1.5">
  <div className="flex items-center gap-2">
    <span className="text-base">{themeIcons[theme.theme] || '📝'}</span>
    <span className="text-sm font-medium text-text capitalize">{theme.theme}</span>
    <span className="text-xs text-text-secondary">({theme.messageCount} mentions)</span>
  </div>
  <span className={`text-[11px] font-medium px-2 py-0.5 rounded-full ${severityColors[theme.severity]}`}>
    {theme.severity}
  </span>
</div>
<p className="text-sm text-text-secondary ml-7">{theme.fixSuggestion}</p>
```

**Changes:**
- Icon size: explicit `text-base` for consistent emoji sizing
- Theme name: add `capitalize` for display quality
- Mention count: `text-muted` to `text-secondary` (better contrast)
- Severity badge: `text-[10px]` to `text-[11px]`, `py-1` to `py-0.5` (less cramped)
- Fix suggestion: `text-xs` to `text-sm`, `text-text/70` to `text-secondary`, `ml-6` to `ml-7` (align with icon after size change)
- `mb-1` to `mb-1.5` (slightly more breathing room)

### Message Cards (Recent Feedback section)

**Current (line 103-109):**
```tsx
<div key={msg.id} className="p-2 bg-white/5 rounded-lg text-xs">
  <div className="flex items-center gap-2 mb-1">
    <span className="font-medium text-text">{msg.userId}</span>
    <span className="text-[10px] px-1.5 py-0.5 rounded bg-white/10 text-muted">{msg.theme}</span>
    <span className={`text-[10px] px-1.5 py-0.5 rounded ${severityColors[msg.severity]}`}>{msg.severity}</span>
  </div>
  <p className="text-text/70">{msg.message}</p>
</div>
```

**After:**
```tsx
<div key={msg.id} className="p-3 bg-surface-alt rounded-lg border border-border-light">
  <div className="flex items-center gap-2 mb-1.5">
    <span className="text-[11px] font-medium px-2 py-0.5 rounded-full bg-white/10 text-text-secondary capitalize">{msg.theme}</span>
    <span className={`text-[11px] font-medium px-2 py-0.5 rounded-full ${severityColors[msg.severity]}`}>{msg.severity}</span>
    <span className="text-[11px] text-text-muted ml-auto">{new Date(msg.createdAt).toLocaleDateString()}</span>
  </div>
  <p className="text-sm text-text-secondary leading-relaxed">{msg.message}</p>
</div>
```

**Changes:**
- **Removed:** `msg.userId` display entirely
- **Added:** `msg.createdAt` rendered as a localized date string, positioned right with `ml-auto`
- Card padding: `p-2` to `p-3` (match theme cards and other panels)
- Card background: `bg-white/5` to `bg-surface-alt border border-border-light`
- Removed `text-xs` from card wrapper (let children control size)
- Theme badge: `text-[10px]` to `text-[11px]`, `px-1.5` to `px-2`, `py-0.5` stays, `rounded` to `rounded-full`, `bg-white/10` to `bg-white/10`, add `capitalize`, add `font-medium`
- Severity badge: `text-[10px]` to `text-[11px]`, `px-1.5` to `px-2`, `rounded` to `rounded-full`, add `font-medium`
- Message text: `text-text/70` to `text-sm text-text-secondary leading-relaxed`
- `mb-1` to `mb-1.5`

const severityColors: Record<string, string> = {

**No change needed.** The severity color mapping is correct and consistent with OrdersPanel and MarketPanel badge patterns.
